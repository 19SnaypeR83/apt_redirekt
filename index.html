    apt Redirekt

    It's a redirektor for apr-urls. If a CMS doesn't support apturl's you can use this service. It fully written in HTML and JS. The URl will look like this: http://thomas131.github.com/apt_redirekt/apt.html?name=gedit You can also download this repo with 

    git clone https://github.com/Thomas131/apt_redirekt.git

    or by downloading https://github.com/Thomas131/apt_redirekt/archive/master.zip. Than the URL's will look different.
